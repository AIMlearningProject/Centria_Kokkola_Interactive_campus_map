# Centria_Kokkola_Interactive_campus_map

Project can be found at master branch.

## What it holds:

- Room search
- Full map including floor switching.
- Room navigation from main doors to specific room includes step by step navigation.
- Reservations handler, that is room specific.
- Real time intergration to all functions, that should have it.

## What it does not have:

  - Does not have real reservation loader. Only test reservations are added to room 131.
 
## How to test?

You can test the project by downloading the content from master branch as zip to your computer. To start the program navigate to folder Centria_Campus_Map_Build_9-2-2026 and click exe file.
